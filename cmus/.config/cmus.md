### cmus Install
```sh
sudo pacman -S cmus
```
### 播放控制
| 快捷键 | 说明           |
|--------|----------------|
| x      | 播放或重播音乐 |
| c      | 暂停           |
| b      | 播放下一曲     |
| z      | 播放上一曲     |
| u      | 更新缓存       |
| q      | 推出程序       |

### 循环模式
| 快捷键   | 说明         |
|----------|--------------|
| s        | 随机播放     |
| f        | 顺序播放     |
| r        | 循环播放     |
| Ctrl + r | 切换单曲循环 |

### 音量调节
| 快捷键 | 说明 |
|--------|------|
| `-`    | 减小 |
| `=`    | 增大 |

### 创建歌单
```sh
:pl-create <name>
```
